# Swantana - Institute Management System

## Project Overview
Swantana is a web-based institute management system developed to streamline trainee registration, administrative management, communication handling, and student record management for training institutes.

The system provides role-based access for administrators and users, allowing efficient handling of trainee applications, contact queries, authentication, and institutional workflows.

---

## Features

- Student application management
- User registration and login system
- Role-based authentication (Admin/User)
- Admin dashboard for managing records
- Contact and query management
- Search and pagination functionality
- Secure password hashing
- Responsive user interface
- MySQL database integration

## Database Components

The project database includes:

- `applications` table
- `users` table
- `contacts` table

Key database functionalities:
- Student data management
- Authentication support
- Admin role handling
- Search operations
- Timestamp-based tracking
